# simple_web_crawler

## 簡易網路爬蟲
# a. 檢查使用者輸入的參數是否有誤，並且輸出適當的說明訊息 
# b. 後顯示狀態 (URL 是否有效) 
# c. 統計資訊(檔案數量、下載容量、下載時間) 
# d. 下載過程顯示各物件的下載狀況 (物件資訊、進度、速度)。
